# neura

a multi-game GSC menu for **IW8** (MW:2019), **S4** (Vanguard), and **IW9** (MW:2022), created by [nyli2b](https://x.com/nyli2b) & [mjkzy](https://x.com/mjkzys)

neura will **ALWAYS** be free and open source.
